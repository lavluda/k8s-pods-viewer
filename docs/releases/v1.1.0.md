# k8s-pods-viewer v1.1.0

`k8s-pods-viewer` v1.1.0 gives the pod view a much stronger operator-focused UI.

## Highlights

- a new context bar shows cluster context, namespace scope, pod count, unhealthy count, grouping mode, sort mode, and current time
- pod rows now use semantic health indicators with clearer visual hierarchy, restart awareness, and stronger warning and critical highlighting
- the main list can be grouped by namespace, workload, or shown flat with the `g` shortcut
- CPU and memory bars now show percentages inside the bar with consistent threshold markers
- `/` opens live filtering across pod, namespace, workload, status, and node names
- `c`, `m`, and `s` sort instantly by CPU, memory, and status
- `i` toggles compact and detailed modes, including compact node aliases for cleaner scanning
- the node pressure panel now uses the same bar language as the pod list for faster comparison

## Release validation

This release was validated with:

- automated Go tests via `go test ./pkg/... ./cmd/...`
- release smoke checks via `make test-release`
- local release build verification via `make build`

## Notes

- pod drill-down actions such as logs, describe, and exec are still future work
- live pod and node usage still depends on `metrics.k8s.io` being available in the cluster

## Install

```bash
go install github.com/lavluda/k8s-pods-viewer/cmd/k8s-pods-viewer@v1.1.0
```
